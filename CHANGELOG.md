# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-23

### Features
- 輸入家庭住址，自動產生專屬防災手冊 PDF（中/英文版）
- 全台 83,000+ 筆避難收容所、防空避難處、醫療院所資料
- 15,000+ AED 自動體外心臟去顫器位置
- 214 間急救責任醫院分級標示（重度/中度/一般）
- 捷運站納入防空避難選項
- 地址定位精度至門牌等級（雙 geocoder 交叉比對）
- 定位不準時可手動「重新定位」，重新查詢附近設施
- 緊急行動卡：4 種災害應變一頁搞定
- 緊急避難卡（信用卡大小）：家人集合點與聯絡方式
- LINE 內建瀏覽器偵測與手動操作提示
- 政府防災資源連結（依縣市）
- QR Code 分享、LINE 分享
- 資料每月自動更新（GitHub Actions）
- 所有資料僅在瀏覽器處理，不會上傳

### Data Sources
- 內政部消防署避難收容處所資料
- 各縣市政府開放資料平台（防空避難處）
- 衛生福利部醫療機構基本資料
- 衛生福利部 AED 急救資源
- 衛生福利部急救責任醫院名單
